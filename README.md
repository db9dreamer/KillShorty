# KillShorty

A simple Chrome browser extension that removes the "Shorts" section from your YouTube subscription feed, providing a cleaner browsing experience.

To install the extension, follow these steps:
1. Clone or download the repository to your local machine.
2. Open your browser and navigate to the extensions page (e.g., `chrome://extensions/` for Chrome).
3. Enable "Developer mode" (usually found in the top right corner).
4. Click on "Load unpacked" and select the directory where you cloned or downloaded the repository.

The extension should now be installed and active.
